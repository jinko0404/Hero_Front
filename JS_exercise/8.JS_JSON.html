JSON

JSON(JavaScript Object Notation)

구분값 JavaScript Object JavaScript Object Notation
Boolean var n = new Boolean(true); var n = true;
Number var n = new Number(3); var n = 3;
String var s = new String("hello"); var s = "hello";
Array var arr = new Array(); arr = [];
Object var obj = new Object(); var obj = {}; 
JSON을 사용하면 모든 데이터형을 서버와 주고 받을 수 있다.(사용목적)

숫자, 문자열, Boolean, 배열, 객체, null, undefiend

'{'로 시작며여 '}'로 끝나며 그 속에 데이터를 표기하고 'key : value(값)' 쌍으로 구성된다.
{
    "name" : "이순신",
    "age" : 27,
    "birth" : "1990-01-01",
    "gender" : "남",
    "married" : true,
    "address" : "서울특별시 종로구",
    "family" : ["아버지", "어머니", "동생"]
}
