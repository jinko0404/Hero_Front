함수(Function)

선언적 함수

반환 값 선언없이 Function 키워드만 이용하여 사용
Function 함수명(){ 소스코드 작성 }으로 작성해서 실행하는 방법

익명함수

function에 함수명을 작성하지 않고 변수에 함수를 할당하는 방법
var 함수명 = function () {소스코드 작성} 으로 작성하여 실행하는 방법

즉시 실행 함수(스스로 실행하는 함수, IIFE)

(function() { 소스코드작성 })(); 으로 작성하여 버튼없이 실행된다.

함수의 전달인자와 매개변수

함수의 리턴처리

매개변수로 함수 전달(콜백함수)

익명함수를 매개변수로 활용
함수도 하나의 자료형이기 때문에 매개변수로도 전달 가능
매개변수로 넘겨지는 함수를 콜백함수라고도 부른다.
함수의 매개변수는 함수도 받을 수 있다.
