삽입 메소드 part1

A : 기존 태그, B : 추가할 태그

$(A).append(B) : B를 A의 내부문자 뒷부분에 추가한다.(자식으로)
$(A).after(B) : B를 A의 뒤에 추가한다.(형제, 자매로)

$(A).prepend(B) : B를 A의 내부문자 앞부분에 추가한다.(자식으로)
$(A).before(B) : B를 A의 앞에 추가한다.(형제, 자매로)
