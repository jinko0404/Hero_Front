객체 탐색(jQuery Traversing)

-jQuery Traversing은 jQuery의 순회(탐색)
-특정요소를 찾거나 필터링하는 작업이며 대부분 선택자(selector)를 사용하지만 1차적으로 선택자에 의해서 찾은 요소들을 다시 2차 필터링 하거나 새로운 요소를 추가할 때에는 Traverse관련 메소드를 사용한다.

filtering 메소드

first() : 선택된 요소 중 가장 첫번째 요소를 리턴
last() : 선택된 요소 중 가장 마지막 요소를 리턴
eq() : 인덱스 번호와 일치하는 요소 리턴(0부터 시작)

Ancestor 메소드

parent() : 선택된 요소의 바로 위 상위 요소만 리턴
parents() : 선택된 요소의 모든 상위 요소 리턴
parentsUntil(인자) : 선택된 요소(초과) 부터 인자요소(미만)까지 범위요소 리턴

Descendants 메소드 : 선택된 요소의 하위 요소들을 선택 할 수 있는 메소드

children([인자])
1. 선택된 요소의 모든 자손(다음 레벨) 객체를 리턴
2. 선택된 요소의 인자와 일치하는 자손(다음 레벨)객체 리턴
find(인자) : 선택된 요소의 인자와 일치하는 모든 후손(모든레벨) 객체 리턴

Siblings 메소드

같은 레벨에 있는 요소(형제)를 선택할 수 있는 메소드

siblings([인자])
1. 선택된 요소와 같은 레벨(형제)에 잇는 요소 리턴
2. 선택된 요소와 같은 레벨(형제) 중 인자와 일치하는 요소리턴 (자기 제외)
next() : 선택된 요소와 같은 레벨(형제)들중 선택된 요소의 다음 한개 요소 리턴
nextAll() : 선택된 요소와 같은 레벨(형제)들 중 선택된 요소의 다음 모든 요소 리턴
nextUntil(인자) : : 선택된 요소와 같은 레벨(형제)들 중 인자까지 범위의 모든 요소들을 리턴
prev() : 선택된 요소와 같은 레벨(형제)들중 선택된 요소의 이전 한개 요소 리턴
prevAll() : 선택된 요소와 같은 레벨(형제)들 중 선택된 요소의 이전 모든 요소 리턴
prevUntil(인자) : 선택된 요소와 같은 레벨(형제)들 중 인자까지 범위의 모든 요소들을 리턴
